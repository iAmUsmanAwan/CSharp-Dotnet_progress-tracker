## Run Locally

Clone the project

```bash
  git clone https://github.com/iAmUsmanAwan/CSharp-Dotnet_progress-tracker.git
```

Go to the project directory

```bash
  cd my-project-folder-name
```

If you have already set up your IDE, then in terminal type
```bash
  dotnet run
```

If you want to write your own code, first create a console
```bash
  dotnet new console
```
